# ML模型训练步骤

## 1. 在 Vision Pro 上收集数据

1. 运行 handtyping 应用
2. 进入 Debug ML Data Collection 界面
3. 点击"开始采集"
4. 对每个手势录制30秒连续视频（自动推进）
5. 完成后点击"导出 JSON"

导出的文件保存在：
```
Documents/MLTrainingData/ml_training_YYYY-MM-DD_HHMMSS.json
```

## 2. 传输数据到 Mac

**方法1：Xcode Devices**
1. 连接 Vision Pro 到 Mac
2. Xcode → Window → Devices and Simulators
3. 选择设备 → 下载容器
4. 找到 handtyping 应用容器
5. 提取 Documents/MLTrainingData/*.json

**方法2：Finder（如果支持文件共享）**
- 通过 Finder 访问设备文件系统

## 3. 在 Mac 上训练模型（自动清理数据）

```bash
cd handtyping
swift run MLTrainingTool <path-to-json>
```

训练过程包含：
- **自动数据清理**：使用IQR方法移除离群帧
- **Transfer Learning**：使用预训练模型
- **数据增强**：旋转、缩放、噪声

生成 `HandGesture.mlmodel` 文件。

## 4. 编译模型

```bash
xcrun coremlcompiler compile HandGesture.mlmodel .
```

生成 `HandGesture.mlmodelc` 目录。

## 5. 部署到 Vision Pro

**方法1：添加到项目**
- 将 `HandGesture.mlmodel` 拖入 Xcode 项目
- Xcode 自动编译并打包

**方法2：手动复制**
- 将 `HandGesture.mlmodelc` 复制到 Documents 目录

## 数据清理说明

训练工具会自动清理：
- 缺失关键关节的帧
- 距离异常的离群帧（IQR方法）
- 保留高质量数据用于训练

## 当前模型配置

- **算法**: Transfer Learning
- **批次大小**: 64
- **迭代次数**: 200
- **数据增强**: 旋转、缩放、噪声
- **数据清理**: 自动IQR离群值检测
